# Docusaurus Setup

This folder is a Docusaurus docs project for the Real-time Event Streaming documentation.

## Install

```bash
cd projects/real-time-event-streaming/_docusaurus
npm install
```

## Run locally

```bash
npm run start
```

## Build

```bash
npm run build
```

The generated static site will be in `build/` and copied to `../docs/` by the build script.
